# Advanced Calculator

A web-based advanced calculator built using HTML, CSS, and JavaScript.  
The application performs both basic and advanced mathematical operations with proper rounding and error handling.

## Features
- Addition, subtraction, multiplication, and division
- Power and n-th root calculations
- Results rounded to two decimal places
- Division-by-zero error handling
- Modular and reusable JavaScript functions
- Clean and simple user interface

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## How It Works
- The user inputs two numbers
- JavaScript functions retrieve and process the inputs
- Results are rounded to two decimal places using `toFixed(2)`
- Errors such as division by zero are handled gracefully

## Example
Number 1 = 10
Number 2 = 3
Division result: 3.33
